                        X-CART LITE PROJECT DOCUMENTATION


X-CART Lite project layout // {{{
==========================

/           - root
index.php   - alias to cart.php
cart.php    - customer front-end executable
admin.php   - administartor backoffice executable
prepend.php - common-used startup executable
welcome.html     - customizable customer front-end static HTML page
shop_closed.html - customizable storefront-is-closed static HTML page
bin/        - executable binaries and scripts
classes/    - project clases code
compat/     - compatibility code
doc/        - user's documentation
etc/        - configuration files
lib/        - third-party classes
share/      - develpment specific documentation & code
skins/      - UI templates
sql/        - SQL database code & data
tests/      - classes and application tests
var/        - runtime templorary storage, compilation and logging folder

// }}}

Project classes code organization // {{{
=================================

classes/        - top-level repositary
    DB/         - PEAR-compatible database layer code
    Log/        - PEAR-compatible logging code
    XLite.php   - the main class file for project 
    base        - common-user project classes
    dialog/     - customer frontend request handlers
    kernel/     - project core component classes
    modules/    - module classes
    view/       - customer UI component classes
    validators/ - input verification code classes
    admin/     - administrator specific backoffice interface classes, including:
        base/
        dialog/
        view/

// }}}

Project configuration files // {{{
===========================

etc/
    config.php       - the default configuration file
    config.local.php - cutomer-specific configuration (overrides default)

// }}}

Project UI templates structure // {{{
==============================

skins/          - top-level skins directory
    default/    - the default customer's UI skin templates
        en/     - the detault skin locale (English)
        *.tpl   - skin templates
    admin/      - administrator UI templates        
        en/     - locale
        *.tpl   - skin templates

// }}}

Project SQL database code // {{{
=========================

sql/
    xlite_tables.sql - code to create SQL database tables
    xlite_data.sql   - the detault SQL data & demo data

// }}}

Project runtime templorary storage // {{{
==================================

var/        - top-level storage directory
    backup/ - directory to save backup/restore sql data
    run/    - runtime skin and classes compilation
    log/    - log file storage
    tmp/    - temporary directory for uploads
    html/   - decorated skins storage

// }}}
